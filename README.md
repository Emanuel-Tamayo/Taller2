## Repositorio

### 1. Crear un repositorio en github.
1. Entrar a https://github.com  
2. Hacer clic en el botón que tiene un **"+"** y dice **"Create new..."**.
3. Hacer clic en **"New repository"**.
4. Escribir el nombre del repositorio (por ejemplo: `programa1`).
5. Poner una breve descripción de para que se piensa usar (opcional).
6. En la configuración elegir si será público o privado.
7. Activar el **README** (ponerlo en On).
8. Dar clic en **Create repository**.

### 2. Principales comandos que vas a necesitar.
1. **git status**: Muestra el estado de los archivos.
2. **git add .**: Agrega todos los archivos para prepararlos para un commit.
3. **git commit -m "Mensaje descriptivo del cambio"**: Guarda un cambio en el historial del proyecto.
4. **git clone "El link del repositorio"**: Para clonar un repositorio existente.
5. **git push -u origin main**: Subir cambios a github.
6. **git pull**: descargar cambios de un repositorio remoto.
